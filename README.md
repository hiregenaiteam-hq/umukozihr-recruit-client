# UmukoziHR-webapp
UmukoziHR Enterprise WebApp

## How to Install Dependencies and Start the Server

1. **Install pnpm (if you don't have it):**
   ```sh
   npm install -g pnpm
   ```

2. **Install dependencies:**
   ```sh
   pnpm install
   ```

3. **Start the development server:**
   ```sh
   pnpm dev
   ```
